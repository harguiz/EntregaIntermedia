# EntregaIntermedia
Descripción: Web de apuestas para pronosticos del mundial 2022. 
Se presentan tres formularios para carga: Datos Personales, Pronosticos (Campeon, Subcampeon y Goleador) y Apuestas (medio de pago y monto).

http://127.0.0.1:8000/App1/inicio

Desarrollo del Contenido de la página. 

http://127.0.0.1:8000/App1/formu1

Carga del formulario de Datos Personales. Bajar hasta la sección Pronosticos. Se encuentra dentro del primer recuadro.

http://127.0.0.1:8000/App1/formu2

Carga del formulario de Pronosticos. Bajar hasta la sección Pronosticos. Se encuentra dentro del segundo recuadro.

http://127.0.0.1:8000/App1/formu3

Carga del formulario de Apuestas. Bajar hasta la sección Pronosticos. Se encuentra dentro del tercer recuadro.

http://127.0.0.1:8000/App1/busqueda

Se encuentra al final de la pagina el recuadro de busqueda. Busca los pronosticos para el equipo Campeón realizados del pais que se selecciona, y muestra también los pronosticos para Sub-Campeón y goleador. Probar con "Argentina".
